# Architecture Decision Records (ADR)

This folder stores ADRs for technical decisions made in the TrustLayer project.

## Naming

Use the format: `NNNN-short-title.md` where `NNNN` is a zero-padded sequence.

Example:
- `0001-adr-process.md`

## Template

```
# ADR NNNN: Title

- Status: Proposed | Accepted | Deprecated | Superseded
- Date: YYYY-MM-DD

## Context
What problem are we solving, and what constraints apply?

## Decision
What did we decide, and why?

## Consequences
What are the trade-offs, follow-up actions, or risks?

## Alternatives Considered
What other options were considered and why were they not chosen?
```
